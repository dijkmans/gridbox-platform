# Gridbox Raspberry Pi Agent

Deze applicatie draait op de Raspberry Pi en stuurt de Gridbox hardware aan.

Taken:
- verwerken van fysieke knop
- aansturen van relais (open, close, licht)
- toepassen van Gridbox-gedrag via gridbox-core
- ophalen van configuratie via het platform

Deze agent bevat geen businesslogica en geen gebruikerslogica.
