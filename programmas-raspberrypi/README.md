# Gridbox Raspberry Pi Agent

Deze agent draait lokaal op de Raspberry Pi en stuurt de Gridbox hardware aan.

Taken:
- aansturen van rolluik (open en dicht)
- aansturen van verlichting
- verwerken van fysieke knop
- communiceren met het Gridbox platform
- lokaal blijven werken bij netwerkuitval

Deze software draait autonoom en bevat geen businesslogica.
